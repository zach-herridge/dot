# General Development Principles

## Investigation & Analysis
- Read entire files to understand full context
- Search smartly in relevant directories
- Gather sufficient context before making changes
- Explore systematically but efficiently

## Code Development
- Plan first for complex tasks
- Search for existing patterns/utils before writing new code
- Don't settle for easier alternatives - achieve the actual requirement
- Follow established patterns and conventions
- Consider maintainability and readability

## Problem Solving
- Be thorough, don't give up easily
- Verify understanding before implementing
- Consider edge cases and error handling
- Think about testing and validation early
